# 新博客开发架构

## 概述

此项目是个人博客搭建的练手项目，也为了日后开发方便，再次我们应用   **python-Django**  框架搭建后端界面，使用**Vue** 框架搭建前端界面。数据库方面使用腾讯云**CynosDB**，用来存储用户信息及博客相关内容信息。
